Atualização de Base

Script responsável por identificar automaticamente o arquivo CSV mais recente (com base na data no nome), carregar seus dados e atualizar uma planilha Excel existente.

Funcionalidades principais:

Limpeza da base anterior Inserção dos novos dados Replicação automática de fórmulas Preservação da estrutura da planilha

Ideal para manter bases sempre atualizadas com praticidade.

Consolidação de Relatório Mensal

Script voltado para a consolidação de dados mensais, integrando informações de múltiplos arquivos CSV (Analítico e Retornos) em um único relatório Excel.

Funcionalidades principais:

Leitura automática de arquivos por data Consolidação de dados de diferentes fontes Tratamento e padronização de informações Organização dos dados em abas específicas Preparação completa do relatório para análise

Indicado para rotinas recorrentes de geração de relatórios operacionais e indicadores.
